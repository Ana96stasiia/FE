1. Команда git revert: создает новый коммит, который вносит изменения, противоположные указанному коммиту.

2. Изменить последний коммит: git commit --amend

3. Git remote rm origin: отключает удаленный репозиторий с именем origin от локального репозитория.

4. Merge conflict: происходит, когда два человека изменяют одни и те же строки в файле или когда один разработчик удаляет файл, который другой разработчик изменяет.

5. Посмотреть отличия между ветками: git diff "first" "second".

6. Удалить локальную ветку: git branch -d "..."; удалить удаленную ветку: git push "repository" --delete "branch".

7. Сделать файл неотслеживаемым: непроиндексировать или не зафиксировать в коммите.
